## How to Run Locally

### 1.Clone the repository
git clone [https://github.com/Asha-08/fire-ai-task.git]

### 2.Install dependencies
npm install

### 3.Start the development server
npm run dev (for vite)
The project will run locally in browser.

### Build for production
npm run build -- generates a dist/ folder for deployment.

### For proper routing on Netlify, a _redirects file is added to ensure all routes point to index.html.
